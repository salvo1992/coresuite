Brt Rest API Tracking
Brt Rest API Tracking / Tracking API / ParcelID (Get)
In questo argomento
    GetParcelID
    In questo argomento

    Resource Path: /rest/v1/tracking/parcelID/{parcelID}

    Uri: https://api.brt.it/rest/v1/tracking/parcelID/{parcelID}

    Descrizione
    Servizio di tracking della spedizione interrogabile con il segnacollo (parcelID) restituito dal servizio di create. 
    Parameters
    NomeDescrizioneData Type
    parcelID

    Tipo Parametro: path parameter

    Identificativo del collo con cui si vuole effettuare il traking della spedizione

    		string
    Request Headers
    NomeDescrizione
    userID

    Codice utente per l'autenticazione al servizio.

    Dato obbligatorio.

    password

    Password utente.

    Dato obbligatorio.

    Response

    The response body is of type parcelIDResult.

    Successful operation

    Sommario

    parcelIDResult
    ttParcelIdResponse : ttParcelIdResponse
    currentTimeUTC : string
    executionMessage : executionMessage
    code : integer

    Codice esito richiesta.

    I messaggi sono codificati con questa logica: 

    • code>0 esito postivo, viene comunque segnalato un warning
    • code=0 esito positivo
    • code<0 esito negativo

      •  da -1 a -60 sono errori comuni a tutti i metodi.

      • da -61 a -99 sono errori dedicati alla scrittura spedizione

      • da -100 a -160 sono errori dedicati alla conferma / cancellazione di una spedizione.
